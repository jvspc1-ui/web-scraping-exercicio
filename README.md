# Exercício de Web Scraping

Este projeto foi desenvolvido para a aula prática com o objetivo de acessar um site com conteúdo informativo e extrair os dados automaticamente.

## Site utilizado
https://quotes.toscrape.com/

## Objetivo
Extrair citações e autores da página inicial e salvar os dados em um arquivo CSV.

## Tecnologias utilizadas
- Python
- Requests
- BeautifulSoup
- CSV

## Como executar o projeto

1. Instalar as dependências:
pip install -r requirements.txt

2. Executar o script:
python scraper.py

O arquivo `citacoes.csv` será gerado automaticamente com os dados extraídos.
